# App Management Component

Handles global application pages and settings that are not specific to a single
domain entity.

## Key Views

- **Settings Page** (`settings_page.dart`): User preferences (Language, Dark
  Mode, Notifications).
- **Install Page** (`install_page.dart`): Instructions and links for installing
  the PWA or Native app.
- **Instance Install Page** (`instance_install_page.dart`): Specialized
  installation flow for specific instances.

## Models

- `language_model.dart`: Supported languages configuration.
